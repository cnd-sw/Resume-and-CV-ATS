How the project works.

PDF_Resume Summariser.ipynb

This file works by taking in the path of the resume file from the local computer.
When run through this algorithm, it will summarise the Resume of PDF format and return the summarised essay in form of text.

Resume Screening(ATS).ipynb

This file works by taking in the main works for the resume.
There are input words of blacklisted words, where if detected in the summary, the resume will be rejected.

Keywords are takin in input where the resume will be shortlisted based on these words.
It means that if the resume consists of these words, the resume will be considered.

Summary of essay is taken as input. 
When the ATS works, it considers this text and the score of ATS is calculated from this.

The working of the models are explained in the Readme.md file
